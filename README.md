## Overview

This project explores digital literacy applications like professional online presence, coding practice, and safe digital habits. It includes practical tasks such
as phishing case studies, portfolio creation, coding challenges, email etiquette, and preventive checklists. The project builds a strong fundamental knowledge and experiencein digital literacy.


## Task Summaries

### Task 1: Prezi Infographic

* Created an interactive infographic on digital literacy.
* Includes link to my Prezi infographic and a brief description.

### Task 2: Portfolio

* Platforms: LinkedIn, GitHub, Kaggle
* Includes screenshots of set-up profiles and reflection notes on purpose and challenges faced.

## Key Learnings

* Understanding digital literacy tools and its applications.
* Developing professional online presence on LinkedIn, GitHub, and Kaggle.
* Hands-on experience with coding, forms, and digital communication.
